app/source/styles/examples/mixins.css

Requires:

  • plugins from previous examples
  • postcss-mixins for mixins

The mixin can accept variables to give it more power.

@define-mixin icon $network { .icon--$(network) { background-image: url(/images/social/$(network).svg); } } .mixins { display: flex; justify-content: space-between; } .icon { flex: 1; width: 40px; height: 40px; margin-top: 0; background-position: center; background-repeat: no-repeat; background-size: contain; }

Now we can quickly create 5 new classes with no repeated code.

@mixin icon twitter; @mixin icon facebook; @mixin icon gplus; @mixin icon instagram; @mixin icon youtube;
